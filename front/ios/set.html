# 基本使用

  • Swift 的所有基本类型（比如 String、Int、Double 和 Bool）默认都是可哈希化的，一个set只接受一种类型
  • 集合没有等价的简化形式。let set: Set<String>
    let s = Set([1, 1, 2, 3, 2, 3, 2])
print(s); // [2, 3, 1]
    1
    2
  • 多种定义方式
    var s3: Set = ["h", "a", "b", "c"];
var s2 = Set(["h", "a", "b", "c"]);
var s1: Set<String> = ["h", "a", "b", "c"];
    1
    2
    3
