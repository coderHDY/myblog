# 阿里云新服务器配置

  • 按量付费云服务器（要先在阿里云充100+余额），普通用法比较便宜
  • 选规格，按最便宜的方式选
  • 选镜像系统，要linux/centos
  • 点击下一步，重要：按量付费
  • 剩下就直接购买，进入安全组配置
  • 进入管理控制台，点开安全组配置
  • 点击配置规则
  • 增加8822端口号（为ssh连接专门配置，也可以设置其他）
  • 配置好远程连接密码，重启
  • 进入远程连接
  • 选择立即登录
  • 编辑ssh配置，打开port为步骤二设置的port
vi /etc/ssh/sshd_config
